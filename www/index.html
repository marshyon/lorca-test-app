message out

message back

click 'SEND MESSAGE TO BACK-END' to populate this field

On application load, the client (browser) triggers from the onload event a call to onStart - a JavaScript function that is registered by and listened for from the server (main go process).

When the 'SEND MESSAGE TO BACK-END' button is clicked, it triggers an event to update the 'Clicked' element (above) and sends a text message comprising of a date time string to the back-end. The back-end returns a date in epoch form and a random name which are rendered in Message Returned.
